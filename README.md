Experimenting with alternative 3D rendering techniques.

A fork of s-macke's [Voxel Space](https://s-macke.github.io/VoxelSpace/VoxelSpace.html), a JavaScript recreation of the terrain rendering algorithm used by *Comanche*.

The goal of this fork is to convert the flying sim controls to a first person character controller, and weave Doom-style billboard sprite entities into the terrain renderer for dynamic entity support.
